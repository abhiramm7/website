TL;DR

A textbook chapter developing the complete analytical framework for linear time-invariant (LTI) state-space systems dx/dt = Ax + Bu, y = Cx + Du. Two central tools — the matrix exponential e^(At) for the initial-condition response and the convolution equation for the forced response — fully characterize system behavior. The chapter closes with Jacobian linearization as a principled reduction of nonlinear dynamics to local linear approximations near equilibria.

Claims

  • The solution to a homogeneous linear system dx/dt = Ax is given by x(t) = e^{At} x(0), where e^{At} is the matrix exponential defined by an infinite series.
  • The input/output response of a linear time-invariant system is fully characterized by the convolution equation, which includes contributions from initial conditions and the impulse response.
  • A linear system's equilibrium point xe = 0 is asymptotically stable if and only if all eigenvalues of the dynamics matrix A have strictly negative real part.
  • Any matrix A can be transformed into Jordan form, and when Jordan blocks corresponding to zero-real-part eigenvalues are not scalar (1x1), the system is unstable.
  • A nonlinear system can be locally approximated by its Jacobian linearization around an equilibrium point, yielding a linear model valid for small perturbations.
